Life is Short

" class="iframe_placeholder">

Fast paced endless first person shooter!

I challenged myself to make and release a game in 4 weeks, and this is the end product of my work. It's a sci-fi endless FPS game developed solo by me (apart from the shotgun model which one of my friends made for me).

If you want to see my development journey, go check out my YouTube channel (https://bit.ly/sk-studios). If the video hits 50 likes I will keep developing the game further, so make sure to smash like and subscribe while you're there to support me!

If I continue development, I'll add things like high scores, online leaderboards, and whatever the community has to suggest. If you have any suggestions chuck a comment on my video and I'll pick a few to implement!

Just a quick note - the web version will take quite some time to load, so if you are on Windows I would highly recommend downloading the executable below.
